Carbon footprint tracker

A web application created with MEAN stack to track carbon emissions over a journey based on factors including distance travelled, transport mode, etc. while mapping out the journey on an interactive map.

The application has a social aspect as well, with live tweets relevant to the topic are streamed onto the app using Apache Kafka and a news API currently being integrated.

Technologies used: Angular, Node.js, Java, MongoDB, Express, ElasticSearch, Apache Kafka, agm-direction, Redis, polyline, Google distance matrix, AWS EC2, Git, Inginx

Recognize anything?

A website created using React and Python to detect faces, food, items and patterns in an image submitted as a URL.

The application makes use of 2 APIs - The ClarifAI API which is external and an OpenCV API written by me in Pythin using Flask and deployed on AWS Elastic Beanstalk. The user may choose either of these and compare/contrast their performance.

Technologies used: React, Flask, Python, OpenCV, Caffe, ClarifAI API, AWS Amplify, Git, Elastic Beanstalk, Font Awesome, Bootstrap, CSS

Distributed file service

A fault tolerant Distributed file system built with Java that facilitates multithreaded access for users to interact with multiple servers to perform CRUD operations on files stored and replicated across these servers.

Program enforces Mutual exclusion, data replication, distributed transaction, fault tolerance, consistency and consensus.

Technologies/concepts used: Java, Load balanceing, Paxos consensus algorithm, Socket programming
